# 🎮 Tic Tac Toe AI – OpenGL & Minimax

A simple yet effective implementation of the classic Tic Tac Toe game, featuring an AI opponent powered by the Minimax algorithm. Built with OpenGL for rendering and GLFW for window management.

---

## 🧠 Features

- AI opponent using the **Minimax algorithm**
- Smooth 2D rendering with **OpenGL 3.3**
- **GLFW** for cross-platform window/input management
- Basic GUI support via **Dear ImGui (Docking branch)**
- Customizable background color through the UI

---

## 🛠️ Dependencies

Make sure you have the following libraries set up:

- [GLFW](https://www.glfw.org/)
- [GLAD](https://glad.dav1d.de/)
- [Dear ImGui (Docking Branch)](https://github.com/ocornut/imgui/tree/docking)
- OpenGL 3.3 compatible environment

> You can get these through submodules or manually place them inside the project directory.
